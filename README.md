# Конвертер изображений

## Описание

Этот проект представляет собой конвертер изображений, который позволяет преобразовывать изображения между различными форматами, такими как BMP, JPEG и PPM. Проект написан на языке C++ и использует стандартные библиотеки,
а также сторонние библиотеки для работы с изображениями, такие как LibJPEG для обработки JPEG-файлов. Основная цель проекта — предоставить простой и эффективный инструмент для конвертации изображений между популярными форматами. 
Основные возможности: Поддержка нескольких форматов изображений, чтение и запись изображений, модульная архитектура, использование стандартных библиотек, кроссплатформенность. Использовал систему сборки CMake. 

## Системные требования 

C++17, Cmake, LibJPEG.
